\subsection{Detailed Description}
Block Cipher Mode Padding Method This class is pretty limited, it cannot deal well with randomized padding methods, or any padding method that wants to add more than one block. For instance, it should be possible to define cipher text stealing mode as simply a padding mode for C\-B\-C, which happens to consume the last two block (and requires use of the block cipher).
